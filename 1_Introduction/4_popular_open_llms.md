# 4. Popular Open LLMs - Study Notes

## Main Idea

Open LLMs are models whose **weights/parameters** are publicly available.

## Key Model Families to Remember

### 1. Meta - Llama

- Meta's **Llama** family is a major open-model line.
- "Open" here refers to released model weights, not necessarily training code.

### 2. Google - Gemma (vs Gemini)

- **Gemini** = Google's proprietary model family (closed weights).
- **Gemma** = Google's open-model family (publicly available weights).
- Important distinction: same company, different openness model.

### 3. DeepSeek

- Became highly visible in early 2025 for strong open models.
- Demonstrated that open models can approach state-of-the-art quality.
- Released weights so models could be run locally (hardware permitting).

### 4. Mistral

- European AI company with both:
  - proprietary models
  - open models with available weights

## Practical Insight

The open-model ecosystem is broad and evolving quickly. The names above are important anchors, but there are many additional model families.

## Open vs Closed Reminder

- **Open model (practical meaning):** downloadable weights, local execution possible.
- **Closed model:** provider-hosted access via app/API only; no public weights.

## 1-Minute Explanation

Popular open LLMs include Meta's Llama, Google's Gemma, DeepSeek models, and some Mistral models.
The crucial point is that open usually means the model weights are available for download and local use,
while closed models like Gemini are accessed through provider services without weight access.

## Memory Hook

**Llama, Gemma, DeepSeek, Mistral = core open-model names to know.**

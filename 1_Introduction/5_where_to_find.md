# 5. Where To Find Open Models - Study Notes

## Primary Discovery Sources

### 1. Official Provider Websites

- Good for direct information about specific model families (for example, Llama or Gemma).
- Useful when you already know which provider you want.

### 2. Hugging Face (huggingface.co)

- De-facto standard catalog for open AI models.
- Includes models from major providers and smaller/newer teams.
- Best place for broad exploration and comparison.

## What You Can Do on Hugging Face

- Browse by:
  - Trending
  - Most downloads
  - Recency (new releases)
  - Likes/popularity
- Filter by model type:
  - LLMs
  - Vision models
  - Image generation models
  - Audio models

## Why Model Cards Matter

Each model page includes a **model card** with practical details such as:

- Capabilities (for example, text-only vs multimodal)
- Context window information
- Parameter count (for example, 27B)
- Parameter/data type details
- Usage examples (often code snippets)
- Provider notes and constraints

## Shareable 1-Minute Explanation

The main place to discover open models is Hugging Face.
It acts like a central catalog where you can explore models from big and small providers, sort by popularity or recency, and inspect model cards for capabilities, size, and usage details.
Official provider websites are useful too, but Hugging Face is the best starting point for broad model discovery.

## Memory Hook
